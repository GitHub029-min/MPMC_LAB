# Microprocessor & Microcontroller Lab Codes

This repository contains assembly language programs for MPMC Lab (CSE 360).

## Programs
1. Input and Output
2. Addition of Two Numbers
3. Subtraction of Two Numbers
4. Multiplication of Two Numbers
5. Division of Two Numbers
6. Sum of N Numbers using Loop
7. Smallest of Two Numbers
8. Smallest of Three Numbers
9. Largest of Three Numbers
10. Simple Pattern Printing
11. implement the following instructions: SHL, SHR, ROL, ROR
